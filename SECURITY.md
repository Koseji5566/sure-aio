# Security Policy

## Supported Versions

Currently only the `latest` tag and primary branch are supported with security updates.

| Version | Supported          |
| ------- | ------------------ |
| latest  | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

If you discover a potential security issue in this project, please notify us immediately by emailing **security@aethereal.dev**. We will review your report within 48 hours and work with you to resolve the issue promptly. Let us know if you want to be credited for discovering the vulnerability.

Please DO NOT create a public GitHub issue for security vulnerabilities.
